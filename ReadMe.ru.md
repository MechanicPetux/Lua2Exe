# Lua2Exe
Простая программа, компилирующая скрипты Lua в исполняемые файлы.
## Использование
`l2e [опции] [модули...] основной_файл`

Доступные опции:
* -o File           Задаёт имя получившегося файла.
* -c Compiler  Задаёт компилятор C. Поддерживаемые компиляторы: "gcc", "clang".
* -s                  Убрать дебаг информацию из exe.
* -dll                Сделать exe зависимой от DLL.
* -a Options    Дополнительные опции, которые нужно послать компилятору C.
* -\[h | ?\]           Показать эту инфу.

> [!IMPORTANT]
> * Порядок файлов важен. Главный файл должен быть последним.
> * Имена модулей должны быть такими же, как и в `require`. То есть *My.lua* это *My*.
> * Так как нет надёжного способа определить все зависимости, для того, чтобы сделать независимый exe, все модули должны быть предоставлены во время компиляции.


## Функции
Сейчас их не много, но есть все базовое:
* Компиляция в независимый exe.
* Компиляция в зависимый от DLL exe.
* Поддержка разных компиляторов. (Типа. Я не тестировал это)

## Зависимости
Для использования всех функций надо:
* Исходный код Lua.
* Lua DLL.
* Компилятор C.
